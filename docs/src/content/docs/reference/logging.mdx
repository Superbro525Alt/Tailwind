---

To import the `logger` module:

```python
import tailwindall.logger as logger
```

## Usage

### Info

To log info to the console use:

```python
import tailwindall.logger as logger

logger.info("Hello World")
```

### Warning

To log a warning to the console use:

```python
import tailwindall.logger as logger

logger.warning("Hello World")
```

### Error

To log an error to the console use:

```python
import tailwindall.logger as logger

logger.error("Hello World")
```

### Debug

To log a debug message to the console use:

```python
import tailwindall.logger as logger

logger.debug("Hello World")
```
